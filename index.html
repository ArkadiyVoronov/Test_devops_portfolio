Testing Pipeline — Основной фокус

01
Cypress
E2E Testing Framework
E2E-тестирование всего пользовательского пути. Тесты пишутся как пользовательские сценарии с time-travel debugging и автоматическими скриншотами при падениях.
∞ tests local
5 users (dashboard)
OSS free tier
02
Keploy
API Test Generation
Автоматическая генерация E2E тестов из реальных API-вызовов. Записывает KTests и KMocks — революция в тестировании без написания тестов вручную.
OSS free tier
API test gen
CI native
03
Percy
Visual Regression
Визуальное тестирование — ловит баги, которые пропускают юнит-тесты. Сравнивает скриншоты компонентов между деплоями с pixel-level точностью.
5K snapshots/mo
∞ projects
∞ team members
04
Bencher
Continuous Benchmarking
Ловит performance-регрессии до деплоя. Интеграция в CI — бенчмарки запускаются на каждый PR и показывают тренд производительности.
∞ public projects
CI integrated
OSS free
05
Checkly
Synthetic Monitoring
Код-фёрст мониторинг на Playwright. Пишешь тесты как код — они же работают как мониторинг продакшена 24/7 из нескольких регионов.
10K API checks/mo
1.5K browser checks
IaC workflow
06
Buildkite
CI/CD + Test Analytics
Test Analytics — отдельный инструмент для анализа падающих тестов. Автоматически определяет flaky tests и показывает тренды по suites.
5K job min/mo
100K test runs/mo
3 users
